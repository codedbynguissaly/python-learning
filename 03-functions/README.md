# 🧩 Functions

## 📖 What are functions?

A function is a reusable block of code that performs a specific task.

Functions help make your code more organized, readable, and easier to maintain.

---

## 📝 Syntax

```python
def function_name():
    # code

function_name()
```

---

## 💡 Example

```python
def greet():
    print("Hello, World!")

greet()
```

### Function with Parameters

```python
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
```

### Function with a Return Value

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```

---

## 📌 Key Concepts

- `def` → Defines a function.
- **Parameters** → Values received by a function.
- **Arguments** → Values passed when calling a function.
- `return` → Sends a value back to the caller.
